# Quick Start

## 1. Clone repository

```C
git clone https://github.com/Brlaney/strapi-next-fullstack.git
```

## 2. Setup the `backend` **Strapi api**

```C
cd backend

# Install dependencies
yarn install

# Build the admin login portal
yarn build --clean

# If using npm
npm i
&&
npm run build -- --clean

# Run local server
yarn develop
```

## 3. Setup the `frontend` **Next.js app**

```C
cd frontend 

# Install dependencies
yarn install

# Run local server
yarn dev

# If using npm
npm i
&&
npm run dev

```
